# Trabajo de Pruebas de Caja Negra – "Subí que te llevo" - Grupo 6

Este repositorio forma parte de un trabajo práctico de pruebas de **caja negra** realizado sobre el sistema **"Subí que te llevo"**, desarrollado en Java.  
El objetivo general del trabajo es ejercer el rol de tester a partir de un sistema ya definido y su documentación asociada.

---

## Desarrollo del trabajo

Para el desarrollo del trabajo se siguieron, de manera general, estos pasos:

- Lectura y comprensión del SRS y de las funcionalidades principales del sistema.
- Identificación de casos de uso y escenarios relevantes a probar.
- Diseño de casos de prueba de caja negra, considerando entradas válidas e inválidas.
- Ejecución de las pruebas sobre la aplicación.
- Registro de resultados y observaciones (comportamientos esperados vs. obtenidos).

---

## Consideraciones

- Las pruebas se basan exclusivamente en la **documentación entregada** (SRS) y en el **anexo de ventanas**
- El conjunto de casos de prueba no busca cubrir todos los escenarios posibles, sino aquellos considerados **más representativos**.
- No se realizaron modificaciones sobre el código fuente original; el foco está puesto en el **comportamiento del sistema tal como fue entregado**.

---

## Conclusion

El trabajo permitió practicar el enfoque de pruebas de caja negra y reforzar la importancia de diseñar casos de prueba a partir de los requerimientos.  
Este repositorio reúne el material generado (casos de prueba, resultados y documentación mínima) como evidencia del proceso realizado.
